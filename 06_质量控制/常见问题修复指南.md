## 内容问题

### 问题1：逻辑混乱，页面缺乏递进关系

**症状**：
- 页面标题看起来是平行罗列
- 不知道为什么这一页后面是那一页
- 缺乏整体结构感

**原因**：
- 没有先规划大纲就直接开始做
- 缺乏"金字塔原理"的指导

**修复步骤**：

1. **列出现有标题**
   ```
   第1页：产品介绍
   第2页：团队介绍
   第3页：客户案例
   第4页：未来规划
   ```

2. **重新归类和排序**（金字塔原理）
   ```
   层级1：我们的核心成果（总）
   层级2：
     - 产品成功上线（分论点1）
     - 客户认可（分论点2）
     - 团队能力提升（分论点3）
   层级3：未来规划（展望）
   ```

3. **重写标题**（体现递进）
   ```
   第1页：2024核心成果概览
   第2页：产品成功上线，用户突破10万
   第3页：获得50+客户认可
   第4页：团队能力显著提升
   第5页：2025年战略规划
   ```

4. **添加过渡页**
   在大章节之间插入"章节封面"，增强结构感。

**参考资料**：`02_模板库/大纲模板_*.md`

---

### 问题2：数据不足，多为定性描述

**症状**：
- "大幅增长""显著提升"等模糊表达
- 缺乏具体数字
- 无法说服人

**原因**：
- 未充分收集数据
- 不知道如何量化

**修复步骤**：

1. **列出所有定性描述**
   ```
   ❌ "业绩大幅增长"
   ❌ "客户满意度提升"
   ❌ "团队能力增强"
   ```

2. **收集原始数据**
   - 查看业务报表、数据库、Excel表
   - 问相关负责人要具体数字

3. **改为量化表达**
   ```
   ✅ "业绩增长150%，从2000万增至5000万"
   ✅ "客户满意度从82%提升至95%"
   ✅ "团队规模从15人扩展至30人"
   ```

4. **添加对比**（增强说服力）
   - 与上期对比："同比增长150%"
   - 与目标对比："超额完成目标20%"
   - 与行业对比："高于行业平均30%"

**快速技巧**：
- 使用AI分析数据：`04_AI提示词库/数据分析提示词.txt`

### 问题3：内容冗长，一页放不下

**症状**：
- 文字密密麻麻
- 字号很小（<20pt）
- 拥挤感强

**原因**：
- 想说的太多，舍不得删
- 没有提炼关键词

**修复步骤**：

1. **应用"3-5原则"**
   - 每页不超过5条bullet points
   - 每条不超过15字

2. **提炼关键词**
   ```
   ❌ 原文："我们在2024年通过持续优化产品功能和用户体验，成功实现了用户数量的大幅增长。"

   ✅ 精简后："产品优化 → 用户增长150%"
   ```

3. **拆分为多页**
   如果确实内容很多，拆成2-3页：
   - 第1页：总结（3条核心要点）
   - 第2-3页：详细展开（每页聚焦1个要点）

4. **使用视觉元素代替文字**
   - 流程用流程图
   - 对比用表格
   - 数据用图表

**AI辅助**：
使用 `04_AI提示词库/内容策划提示词.txt` 中的"文案精炼"提示词。

## 视觉问题

### 问题4：配色混乱或不一致

**症状**：
- 一页用蓝色，另一页用橙色
- 渐变不自然，有色阶
- 颜色刺眼或过于暗淡

**原因**：
- 随意选色，未使用统一方案
- 不了解配色原理

**修复步骤**：

1. **选择一套配色方案**
   打开 `03_视觉规范/配色方案.json`，选择：
   - 方案A（科技蓝）：技术、创新类
   - 方案B（商务橙）：商务、市场类
   - 方案C（专业绿）：运营、可持续类

2. **全局替换颜色**
   - 标题：使用渐变主色
   - 数据：使用主色或辅色
   - 正文：使用深灰（#2C3E50或#34495E）
   - 背景：白色或浅灰

3. **修复渐变色阶**（PowerPoint）
   - 选中渐变对象 → 右键"设置形状格式"
   - 渐变填充 → 类型选"线性" → 方向135度
   - 光圈设置：
     - 左光圈：主色起点（如#4A90E2），位置0%
     - 右光圈：主色终点（如#5C6BC0），位置100%
     - 勾选"平滑渐变"

4. **批量检查**
   - 缩小视图到30%，浏览全部页面
   - 找出"跳戏"的页面，逐一修改

**工具推荐**：
- Adobe Color（生成和谐配色）
- ColorZilla（浏览器插件，吸取网页颜色）

---

### 问题5：排版拥挤或对齐不整齐

**症状**：
- 元素贴边，无留白
- 文字和图表未对齐
- 元素位置高低不一

**原因**：
- 未使用对齐工具，靠目测
- 不了解留白的重要性

**修复步骤**：

1. **打开对齐辅助功能**（PowerPoint）
   - 视图 → 勾选"网格线"和"参考线"
   - 排列 → "对齐" → 勾选"对齐网格"

2. **设置页边距**
   - 在页面四周留出≥10%的边距
   - 使用参考线标出安全区域

3. **批量对齐**
   - 选中多个元素 → 排列 → 对齐 → 左对齐/居中对齐
   - 选中多个元素 → 排列 → 分布 → 横向分布/纵向分布

4. **增加留白**
   - 删除不必要的装饰元素
   - 增大元素间距（≥行高）
   - 确保页面空白区域≥30%

**对齐检查法**：
画一条垂直参考线，所有左对齐的元素应吸附在这条线上。

### 问题6：字号不合理或不统一

**症状**：
- 标题和正文字号差不多，无层次感
- 最小字号<20pt，投影看不清
- 不同页面字号不一致

**原因**：
- 未预设字体样式
- 随意调整字号

**修复步骤**：

1. **制定字号标准**（参考 `03_视觉规范/字体规范.md`）
   ```
   封面标题：72-96pt
   页面标题：48-54pt
   核心数据：60-96pt
   小标题：32-36pt
   正文：24-28pt
   注释：20-22pt
   ```

2. **创建母版**（PowerPoint）
   - 视图 → 幻灯片母版
   - 设置标题样式：字体、字号、颜色
   - 设置正文样式：字体、字号、行距
   - 保存母版

3. **批量应用母版**
   - 开始 → 版式 → 选择对应版式
   - 自动应用母版样式

4. **手动调整特殊页**
   - 数据页：数字字号增大至60pt+
   - 图表页：标注字号≥20pt

**快速检查**：
3米外看屏幕，如果看不清某些文字，说明字号太小。

## 数据图表问题

### 问题7：图表类型选择不当

**症状**：
- 用饼图展示趋势（应该用折线图）
- 用折线图展示占比（应该用饼图）
- 图表难以理解

**原因**：
- 不了解各类图表的适用场景

**修复步骤**：

1. **识别数据类型**
   - 时间序列 → 折线图或面积图
   - 分类对比 → 柱状图
   - 占比结构 → 饼图或环形图
   - 相关性 → 散点图
   - 多维度 → 雷达图或气泡图

2. **重新生成图表**
   使用 `05_自动化工具/generate_chart.py`：
   ```bash
   # 趋势用折线图
   python generate_chart.py --type line --data "Q1:85,Q2:92,Q3:88,Q4:95"

   # 对比用柱状图
   python generate_chart.py --type bar --data "产品A:30,产品B:25,产品C:20"

   # 占比用饼图
   python generate_chart.py --type pie --data "渠道A:40,渠道B:35,渠道C:25"
   ```

3. **或使用AI推荐**
   使用 `04_AI提示词库/数据分析提示词.txt`，让AI推荐图表类型。

**图表选择速查表**：

| 数据特征 | 推荐图表 | 示例 |
|---------|---------|------|
| 随时间变化 | 折线图、面积图 | 月度GMV趋势 |
| 分类对比 | 柱状图、分组柱状图 | 各产品销量对比 |
| 部分占整体 | 饼图、环形图 | 营收结构占比 |
| 两变量关系 | 散点图 | 投入vs产出 |
| 多维度评价 | 雷达图 | 能力评估 |
| 流程步骤 | 瀑布图、桑基图 | 转化漏斗 |

### 问题8：图表过于花哨或复杂

**症状**：
- 3D效果、阴影、过多装饰
- 网格线密集
- 颜色过多
- 难以快速理解

**原因**：
- 使用Excel/PPT默认样式（通常比较花哨）
- 追求"炫酷"而非"清晰"

**修复步骤**：

1. **去除3D效果**（PowerPoint图表）
   - 选中图表 → 图表设计 → 更改图表类型
   - 选择"2D"版本（如"二维柱形图"）

2. **简化网格线**
   - 选中图表 → 添加图表元素 → 网格线
   - 仅保留主要横向网格线
   - 颜色改为浅灰（#F0F0F0）

3. **减少配色**
   - 单系列：使用主色渐变（深→浅）
   - 多系列：仅使用主色+辅色（2个颜色）

4. **去掉边框和阴影**
   - 选中图表区域 → 设置图表区域格式
   - 边框：无线条
   - 阴影：无阴影

5. **重新生成**（推荐）
   使用 `05_自动化工具/generate_chart.py` 自动生成符合规范的图表。

**黄金原则**：能删则删，图表越简洁越好。

### 问题9：数据标签缺失或不清晰

**症状**：
- 图表上没有数值标签，需要自己估算
- 标签位置不合理，遮挡数据
- 标签字号过小

**原因**：
- 使用默认设置，未添加数据标签

**修复步骤**：

1. **添加数据标签**（PowerPoint图表）
   - 选中图表 → 添加图表元素 → 数据标签
   - 选择位置：
     - 柱状图：置于上方
     - 折线图：置于数据点上方
     - 饼图：置于扇区内或外侧

2. **调整标签格式**
   - 选中数据标签 → 右键"设置数据标签格式"
   - 字号：≥20pt
   - 颜色：深灰或主色
   - 字重：粗体（Bold）

3. **添加单位**
   - 编辑数据标签 → 添加"%"、"万元"等单位

4. **突出关键点**
   - 最大值/最小值用辅色高亮
   - 或用形状标注

**Python生成时**：
在 `generate_chart.py` 中已自动添加数据标签，无需手动操作。

## 细节问题

### 问题10：错别字和语病

**症状**：
- 文字有拼写错误
- 句子不通顺
- 标点符号错误

**原因**：
- 未仔细校对
- 复制粘贴时遗留

**修复步骤**：

1. **使用拼写检查**（PowerPoint）
   - 审阅 → 拼写检查
   - 逐个修正

2. **人工逐页检查**
   - 打印或导出为PDF
   - 用红笔标注错误
   - 逐一修改

3. **AI辅助校对**
   - 复制文字到Claude/GPT-4
   - 提示词："请检查以下文字是否有错别字和语病"

4. **请同事帮忙**
   - 旁观者清，容易发现自己忽略的错误

**高发区域**：
- 数字（如"5000万"写成"500万"）
- 标点（中英文标点混用）
- 专业术语（如"GMV"写成"GMB"）

### 问题11：图片模糊或拉伸变形

**症状**：
- 图片不清晰，像素化
- 图片被拉长或压扁

**原因**：
- 插入低分辨率图片
- 直接拖拽图片边角，未锁定比例

**修复步骤**：

1. **替换高清图片**
   - 原图分辨率≥300dpi
   - 如无高清图，使用AI放大工具（如Topaz Gigapixel）

2. **正确插入图片**
   - 插入 → 图片 → 选择文件
   - 按住Shift键拖拽边角（锁定比例）
   - 或右键"设置图片格式" → 大小 → 勾选"锁定纵横比"

3. **裁剪而非缩放**
   - 如需调整构图，使用"裁剪"功能
   - 避免拉伸

4. **图片压缩设置**（PowerPoint）
   - 文件 → 选项 → 高级 → 图像大小和质量
   - 取消勾选"压缩文件中的图像"
   - 或选择"不压缩"

### 问题12：Logo、页码位置不统一

**症状**：
- 有的页面有Logo，有的没有
- Logo位置不一致
- 页码格式不统一

**原因**：
- 手动逐页添加，未使用母版

**修复步骤**：

1. **使用母版统一添加**（PowerPoint）
   - 视图 → 幻灯片母版
   - 在母版上插入Logo（如右下角）
   - 插入 → 页眉和页脚 → 勾选"幻灯片编号"
   - 关闭母版视图

2. **批量应用**
   - 所有页面自动显示Logo和页码

3. **例外页面**（如封面、章节封面）
   - 创建单独的母版版式
   - 不显示Logo和页码

**Logo位置建议**：
- 右下角（最常见）
- 左下角
- 居中下方

**页码格式建议**：
- 简洁：仅数字（如"1", "2"）
- 分数式："1/20"（当前页/总页数）
- 带装饰："— 1 —"

## 技术问题

### 问题13：PPT文件过大，发送困难

**症状**：
- 文件大小>50MB
- 邮件发不出去
- 打开卡顿

**原因**：
- 插入了大量高清图片
- 未压缩图片

**修复步骤**：

1. **压缩图片**（PowerPoint）
   - 选中任意图片 → 图片格式 → 压缩图片
   - 选择"电子邮件（96 ppi）"或"打印（220 ppi）"
   - 勾选"应用于此文件中的所有图片"
   - 点击"确定"

2. **删除裁剪区域**
   - 压缩图片窗口中 → 勾选"删除图片的裁剪区域"

3. **转换为PDF**（终极方案）
   - 文件 → 另存为 → PDF
   - PDF文件通常更小，且保持视觉效果

4. **使用云盘分享**
   - 上传到百度网盘/Google Drive
   - 分享链接

**预防措施**：
- 插入图片前先压缩（使用TinyPNG等工具）
- 避免插入原始相机照片（通常5-10MB）

### 问题14：中文字体在别人电脑上显示为方框

**症状**：
- 在自己电脑上正常，发给别人后乱码
- 特殊字体变成宋体

**原因**：
- 对方电脑没有安装你使用的字体

**修复步骤**：

1. **嵌入字体**（PowerPoint）
   - 文件 → 选项 → 保存
   - 勾选"在文件中嵌入字体"
   - 选择"嵌入所有字符"
   - 保存文件

2. **使用系统自带字体**（最安全）
   - Windows：微软雅黑、Arial
   - macOS：苹方、Helvetica
   - 跨平台：思源黑体（需单独安装）

3. **转换为PDF**
   - 字体会自动嵌入PDF，无兼容问题

**最佳实践**：
- 团队协作时，统一使用某一字体
- 或制作PPT模板，嵌入字体

---

### 问题15：动画播放卡顿

**症状**：
- 动画不流畅
- 放映时有延迟

**原因**：
- 动画效果过多或过于复杂
- 图片分辨率过高

**修复步骤**：

1. **简化动画**
   - 删除不必要的动画
   - 每页动画不超过2个

2. **减少动画时长**
   - 动画 → 动画窗格
   - 设置时长为0.5-1秒（不宜过长）

3. **优化图片**
   - 压缩图片（见问题13）

4. **关闭硬件加速**（PowerPoint）
   - 文件 → 选项 → 高级 → 显示
   - 勾选"禁用硬件图形加速"
   - 重启PowerPoint

**建议**：
- 重要汇报前，在实际投影仪上预演一次

**最佳实践**：
1. **使用模板**：从标准化模板开始，减少问题
