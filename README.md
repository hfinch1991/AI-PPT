# AI PPT 导演系统

> 一套成熟的、可批量输出高端PPT的完整方法论和工具包

## 项目简介

AI PPT 导演系统是一个标准化的AI辅助PPT制作流程，旨在帮助团队快速产出**高端、专业、视觉统一**的商务演示文稿。

**适用场景**：
- 年终汇报
- 工作汇报
- 项目推进汇报
- 领导审阅材料

**设计风格**：
- 明亮、鲜艳的渐变色调
- 层次分明、高端大气
- 实用不浮夸

## ✨ 最新更新 (v2.0 - 2025-12-18)

基于18张真实高端PPT案例的深度分析,全面升级:

### 新增内容
1. **6章节黄金结构** - 经过验证的15%-30%-20%-15%-10%-10%权重分配
2. **3套新模板** - 工作成果展示型、年度个人述职、年终向上汇报
3. **2个新配色方案** - 高端金蓝(方案D)、多层次专业(方案E)
4. **5种视觉样式** - 波浪线连接、时间轴、半圆图、菱形对比图、环形展示
5. **增强版AI提示词** - 包含具体可视化样式、可运行代码模板、PPT导向数据分析

### 核心亮点
- ✅ 所有模板基于真实案例提取,非理论构建
- ✅ 提示词包含具体页面设计(不只说"图表",而是"波浪线连接的卡片流程")
- ✅ Python代码模板可直接运行,生成高端商务图表
- ✅ 配色方案扩展至5套,覆盖更多应用场景

## 快速开始

### 5分钟上手

1. **采集需求**：填写 `02_模板库/需求采集模板.md`
2. **生成大纲**：使用 `04_AI提示词库/内容策划提示词.txt` 让AI生成结构
3. **选择配色**：从 `03_视觉规范/配色方案.json` 选择一套方案
4. **生成图表**：运行 `05_自动化工具/generate_chart.py`
5. **质量检查**：对照 `06_质量控制/检查清单.xlsx` 逐项核对

详细步骤请查看：[快速开始指南](01_使用手册/快速开始指南.md)

## 核心方法论

本系统采用**5阶段流程**：

```
需求输入 → 内容策划 → 视觉设计 → AI生成 → 质量把控
```

每个阶段都有：
- 标准化的输入输出
- AI提示词模板
- 质量检查标准

详细说明请阅读：[完整方法论](01_使用手册/完整方法论.md)

## 技术栈

- **AI模型**：Claude / GPT-4（内容生成）、DALL-E / Midjourney（视觉素材）
- **数据可视化**：Python + Matplotlib / Plotly
- **PPT自动化**：python-pptx / Google Slides API
- **版本控制**：Git（推荐）

## 使用建议

1. **首次使用**：完整阅读方法论，熟悉流程
2. **日常使用**：直接使用模板和提示词库
3. **团队协作**：统一使用同一套配色方案
4. **持续优化**：每次使用后收集反馈，迭代提示词
