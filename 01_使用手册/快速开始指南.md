# 快速开始指南

> 5分钟掌握AI PPT导演系统的基本使用方法

## 使用流程图

```
第1步：需求采集（5分钟）
   ↓
第2步：AI生成大纲（2分钟）
   ↓
第3步：选择视觉方案（1分钟）
   ↓
第4步：内容填充 + AI辅助（根据页数而定）
   ↓
第5步：质量检查（5分钟）
```

---

## 第1步：需求采集

打开 `02_模板库/需求采集模板.md`，填写以下核心信息：

**必填项**：
- 场景类型：年终汇报 / 工作汇报 / 项目推进 / 领导审阅
- 汇报主题：一句话概括（如"2024年度产品部工作总结"）
- 关键数据：3-5个核心指标（如"GMV增长150%"）
- 目标受众：高层领导 / 同级 / 客户

**选填项**：
- 页数要求
- 特殊需求（品牌色、动画等）

**输出**：完成的需求清单文件

---

## 第2步：AI生成大纲

### 操作步骤

1. 打开 `04_AI提示词库/内容策划提示词.txt`
2. 复制模板，替换 `{占位符}` 为实际需求
3. 提交给Claude或GPT-4
4. 保存AI返回的大纲到 `我的项目_大纲.md`

### 示例

**输入提示词**：
```
你是一位资深的PPT策划专家。根据以下需求，设计一份高端商务PPT大纲：

场景：年终汇报
受众：公司高层领导
核心信息：
- 2024年GMV达到5000万，同比增长150%
- 成功上线3个重点产品
- 团队规模从15人扩展到30人
页数要求：20-25页

请输出：
1. 每一页的标题
2. 每一页的核心内容要点（3-5条）
3. 建议的可视化形式
```

**AI输出示例**：
```
第1页：封面 - "2024年度产品部成果汇报"
第2页：目录 - 四大章节概览
第3页：年度亮点 - 三大核心成果（数字突出）
  - GMV突破5000万（+150%）
  - 产品矩阵扩充至8个
  - 团队能力提升显著
  建议可视化：仪表盘风格的数据卡片
...
```

---

## 第3步：选择视觉方案

### 操作步骤

1. 打开 `03_视觉规范/配色方案.json`
2. 浏览预设的3套配色方案：
   - **方案A - 科技蓝**：适合技术类、创新类汇报
   - **方案B - 商务橙**：适合商务拓展、市场类汇报
   - **方案C - 专业绿**：适合运营、可持续类汇报
3. 查看 `03_视觉规范/参考案例/` 中的效果图
4. 记录选定的配色代码

### 配色示例

选择**方案A - 科技蓝**：
```
主色：#4A90E2 → #5C6BC0（渐变）
辅色：#FF6B6B
文字色：#2C3E50
背景色：#ECF0F1
```

## 第4步：内容填充 + AI辅助

### 4.1 文案优化

对于每一页的内容，使用AI提示词：

```
请优化以下PPT文案，要求：
- 精简至15字以内
- 使用量化表达
- 语气专业有力

原文：我们在今年成功完成了多个项目的交付，客户满意度有所提升。

优化后：成功交付12个项目，客户满意度提升至95%
```

### 4.2 图表生成

**方法一：Python脚本（推荐）**
```bash
cd 05_自动化工具
python generate_chart.py --type bar --data "Q1:85,Q2:92,Q3:88,Q4:95" --color blue
```

**方法二：手动使用提示词**
打开 `04_AI提示词库/图表生成提示词.txt`，让AI生成Python代码

### 4.3 装饰元素生成

使用DALL-E或Midjourney生成背景：

```
Create a modern, minimalist background graphic for a business presentation slide.
Style: bright gradient from #4A90E2 to #5C6BC0, clean geometric shapes,
layered depth effect, high-end corporate aesthetic, 30% opacity maximum,
lots of white space. No text. 16:9 aspect ratio.
```

## 第5步：质量检查

### 使用检查清单

打开 `06_质量控制/检查清单.xlsx`，逐项检查：

**必查项**（10项）：
- [ ] 逻辑性：页面之间有递进关系
- [ ] 数据准确：所有数字已核对
- [ ] 视觉一致：配色/字体统一
- [ ] 层次感：主次分明
- [ ] 留白充足：无拥挤感
- [ ] 渐变自然：过渡流畅
- [ ] 严格对齐：所有元素对齐网格
- [ ] 字号合理：最小≥20pt
- [ ] 动画克制：仅用于强调
- [ ] 5秒可读：快速抓住核心信息

### AI辅助审查

对关键页面截图，使用提示词：
```
你是PPT质量专家，请评分（1-10）并给出改进建议。
评价维度：视觉冲击力、信息传达效率、专业度。
[上传截图]
```

## 常见场景快速操作

### 场景1：紧急汇报（1小时内出稿）

1. 使用最简大纲模板（10页标准结构）
2. 选择已验证的配色方案A
3. 使用纯文字+简单柱状图（跳过装饰元素）
4. 重点做好第1页、第3页、最后1页

### 场景2：重要汇报（精打细磨）

1. 完整走一遍5阶段流程
2. 每页都用AI优化3轮
3. 所有图表用Python生成（确保视觉统一）
4. 请同事预审，收集反馈

### 场景3：批量制作（团队协作）

1. 由负责人统一填写需求模板
2. 分配大纲的不同章节给成员
3. 统一使用同一配色方案
4. 最后由一人汇总并统一调整格式
