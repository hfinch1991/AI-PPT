# 常见问题 FAQ

## 使用流程相关

### Q1: 我是新手，从哪里开始？
**A**: 按以下顺序学习：
1. 先看 [快速开始指南.md](快速开始指南.md)（5分钟）
2. 用一个简单项目实践一遍完整流程（1小时）
3. 查看参考案例，理解"高端感"的标准
4. 根据需要深入学习某个环节

### Q2: 我没有编程基础，能用吗？
**A**: 可以！分两种使用模式：
- **基础模式**：只用模板和AI提示词，手动制作PPT（无需编程）
- **高级模式**：使用Python脚本自动生成图表（需要基本的命令行操作）

大部分功能在基础模式下已足够使用。

### Q3: 一份PPT大概需要多长时间？
**A**: 取决于页数和精细度：
- **快速版**（10-15页）：1-2小时
- **标准版**（20-25页）：3-4小时
- **精品版**（30+页）：6-8小时

熟练后效率会显著提升。

## AI使用相关

### Q4: 应该用Claude还是GPT-4？
**A**: 各有优势：
- **Claude**：擅长逻辑梳理、大纲规划、长文本分析
- **GPT-4**：擅长创意文案、多轮对话优化
- **建议**：内容策划用Claude，文案打磨用GPT-4

### Q5: AI生成的内容不符合预期怎么办？
**A**: 三步优化法：
1. **检查提示词**：是否提供了足够的上下文？
2. **分步骤提问**：把复杂任务拆解成小步骤
3. **提供示例**：给AI一个"好的案例"作为参考

示例：
```
❌ 差的提示词："帮我写一段PPT文案"
✅ 好的提示词："请将以下数据改写成适合PPT的文案（要求：
   - 不超过15字
   - 突出150%的增长率
   - 语气自信专业
   数据：2024年GMV从2000万增长到5000万）"
```

### Q6: AI生成的图表代码报错怎么办？
**A**: 常见问题和解决方案：
- **中文乱码**：检查字体设置，使用"PingFang HK"或"SimHei"
- **导入错误**：运行 `pip install -r requirements.txt` 安装依赖
- **数据格式错误**：确认数据是否为数字类型（非字符串）

如仍无法解决，可以截图错误信息，让AI帮忙修复代码。

## 设计风格相关

### Q7: 什么是"高端感"？如何判断？
**A**: 高端感的5个标准：
1. **留白充足**：页面至少30%是空白
2. **配色克制**：主色不超过3个，渐变自然
3. **层次清晰**：标题>数据>说明，大小对比明显
4. 4. **对齐严格**：所有元素都对齐网格线
5. **字体专业**：使用现代无衬线字体，避免花哨字体

**判断方法**：5秒测试——页面缩小到30%，能否立即看懂核心信息？

### Q8: 配色方案不适合我的品牌怎么办？
**A**: 自定义配色3步骤：
1. 提取品牌主色（通常是Logo的主色）
2. 使用在线工具生成配色方案：
   - [Adobe Color](https://color.adobe.com)
   - [Coolors](https://coolors.co)
3. 更新 `03_视觉规范/配色方案.json`，添加新方案

**注意**：保持"明亮、鲜艳、渐变"的原则。

### Q9: 参考案例在哪里？
**A**: 目前 `03_视觉规范/参考案例/` 文件夹为空，建议：
1. 每次做完优秀的PPT，截图保存到此文件夹
2. 从网上收集高端PPT案例（如站酷、Behance）
3. 建立团队共享的案例库

## 工具使用相关

### Q10: Python脚本怎么运行？
**A**:
```bash
# 第1步：安装Python（如果没有的话）
# 访问 python.org 下载安装

# 第2步：安装依赖包
cd 05_自动化工具
pip install -r requirements.txt

# 第3步：运行脚本
python generate_chart.py --help  # 查看帮助
python generate_chart.py --type bar --data "Q1:85,Q2:92"  # 生成柱状图
```

### Q11: 可以用Google Slides或Keynote吗？
**A**: 可以！方法论适用于所有PPT工具：
- **PowerPoint**：完全兼容
- **Google Slides**：适合团队协作，部分自动化工具可用API
- **Keynote**：视觉效果最佳，但自动化受限

Python脚本生成的图表（PNG格式）可插入任何工具。

### Q12: 如何批量生成多份PPT？
**A**: 两种方案：
- **方案1（手动）**：为每个项目复制一份模板，逐个填充
- **方案2（自动化）**：使用 `ppt_builder.py` 脚本 + 配置文件批量生成

方案2需要一定的Python基础，建议先掌握手动流程。

## 团队协作相关

### Q13: 如何在团队中推广这套方法？
**A**: 建议步骤：
1. **培训会议**（1小时）：讲解方法论 + 现场演示

1. **强制使用检查清单**：每份PPT必须过10项检查
2. **同行评审**：重要汇报需要至少1人预审
3. **建立案例库**：好的案例入库，新人学习有参照

## 高级技巧

### Q16: 如何让AI生成更具创意的设计？
**A**: 提示词技巧：
```
在基础要求上，增加：
- "参考Apple发布会风格"
- "借鉴极简主义设计"
- "融入科技感元素（如电路板纹理、3D几何体）"

示例：
"生成一个仪表盘式数据展示，参考Tesla车载界面的科技风格，
使用蓝色渐变，具有未来感。"
```

### Q17: 如何处理大量数据的可视化？
**A**: 数据简化原则：
- **Top 5原则**：只展示最重要的5个数据点
- **对比突出**：用颜色强调最高/最低值
- **分组归类**：把相似数据合并成一类

工具推荐：
- 复杂图表用 Plotly（交互式）
- 关系图用 pyecharts（桑基图、关系图）

### Q18: 动画效果怎么控制？
**A**: 动画使用原则：
- **克制**：每页不超过2个动画
- **目的明确**：只用于引导视线或强调重点
- **速度适中**：0.5-1秒为宜，避免过快或过慢

建议动画类型：
- 标题：淡入（Fade In）
- 数据：擦除（Wipe）或放大（Zoom）
- 图表：逐条出现（Appear）
