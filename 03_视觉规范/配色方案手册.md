## 配色原则

### 1. 核心设计理念
- **明亮感**：主色明度（HSB的B值）≥ 80%
- **鲜艳感**：主色饱和度（HSB的S值）≥ 70%
- **渐变感**：主色区使用线性/径向渐变，避免纯平涂
- **层次感**：前景-中景-背景有明确的色彩区分
- **克制性**：每套方案主色不超过3个，辅色1-2个

### 2. 使用规则
- **标题**：使用渐变主色，字号60-72pt
- **数据强调**：使用渐变主色或纯色主色
- **正文**：使用深灰色（#2C3E50或#2D3436）
- **背景**：使用浅灰色（#ECF0F1或#F7F9FA）或白色
- **装饰元素**：使用主色渐变，透明度20-40%

## 方案A - 科技蓝

### 适用场景
- 技术类汇报（如技术架构、产品研发）
- 创新类项目（如AI、大数据项目）
- 需要传达"专业、创新、科技"的场景

### 色值定义

#### 主色组（渐变对）
```
主色起点：#4A90E2（蓝）
  - RGB: (74, 144, 226)
  - HSB: (210°, 67%, 89%)

主色终点：#5C6BC0（蓝紫）
  - RGB: (92, 107, 192)
  - HSB: (231°, 52%, 75%)

使用方式：线性渐变，45-135度角
```

#### 辅色（强调用）
```
辅色：#FF6B6B（活力红）
  - RGB: (255, 107, 107)
  - HSB: (0°, 58%, 100%)

用途：数据对比中的负向指标、警示信息、特别强调
```

#### 中性色
```
深灰（文字）：#2C3E50
  - RGB: (44, 62, 80)
  - 用于正文、说明文字

浅灰（背景）：#ECF0F1
  - RGB: (236, 240, 241)
  - 用于卡片背景、页面底色

白色：#FFFFFF
  - 用于主背景、卡片前景
```

### 使用示例

**标题渐变**：
```css
background: linear-gradient(135deg, #4A90E2 0%, #5C6BC0 100%);
```

**数据卡片**：
- 背景：白色或浅灰
- 数字：渐变主色（超大号）
- 增长率：辅色（红色表示警示，绿色表示正向）

**页面布局**：
- 顶部banner：主色渐变（20%透明度）
- 内容区：白色背景
- 装饰图形：主色渐变（30%透明度）

## 高级配色技巧

### 1. 渐变角度选择
```
45度：标准商务风格
90度（垂直）：稳重、正式
135度：现代、动感
径向渐变：聚焦、突出中心
```

### 2. 透明度使用
```
100%（不透明）：标题、核心数据
60-80%：次要文字
30-50%：装饰元素
10-20%：背景层
```

### 3. 色彩层次营造
```
前景：纯色或高饱和度渐变
中景：中等透明度的色块
背景：低饱和度、高明度的浅色
```

### 4. 数据可视化配色
```
单系列图表：主色渐变（深→浅）
多系列对比：主色 + 辅色
趋势图：单色渐变填充
正负对比：绿（正）+ 红（负）
```

## 品牌定制指南

### 如何基于品牌色创建新方案

#### Step 1：提取品牌主色
- 从品牌Logo或VI系统中提取主色（通常1-2个）
- 确保主色符合"明亮、鲜艳"原则（如果不符合，需调亮）

#### Step 2：生成渐变对
**方法1 - 色相偏移**：
- 在HSB色轮上，将主色色相（H）偏移10-30度
- 保持饱和度（S）和明度（B）相近

**方法2 - 明度变化**：
- 保持色相（H）和饱和度（S）不变
- 将明度（B）降低10-20%

**推荐工具**：
- Adobe Color：https://color.adobe.com
- Coolors：https://coolors.co

#### Step 3：选择辅色
- 选择与主色形成对比的颜色（色轮上相隔120-180度）
- 或选择同色系的冷暖对比色

#### Step 4：测试与调整
- 将配色应用到实际PPT页面
- 检查可读性（文字对比度 ≥ 4.5:1）
- 确认符合"高端感"标准

#### Step 5：文档化
- 更新 `配色方案.json`
- 截图保存到 `参考案例/`

## 常见问题

### Q1：配色太鲜艳，不够商务怎么办？
**A**：降低饱和度至50-60%，提高明度至85-90%，形成"莫兰迪商务色"

### Q2：如何判断配色是否"高端"？
**A**：检查清单：
- [ ] 留白充足（≥30%）
- [ ] 渐变自然（无色阶断层）
- [ ] 对比适中（不刺眼）
- [ ] 配色不超过3个主色
- [ ] 文字清晰可读

### Q3：可以混用不同方案吗？
**A**：不建议。同一份PPT应统一使用一套方案，确保视觉一致性。

### Q4：客户要求用品牌色，但品牌色是深蓝色（暗色）怎么办？
**A**：创建两个版本：
- 深色版：保留品牌色，用于Logo和关键元素
- 浅色版：将品牌色调亮30%，用于大面积渐变

## 附录：色彩理论速查

### 色彩心理学
| 颜色 | 传达情绪 | 适用场景 |
|------|---------|---------|
| 蓝色 | 专业、可靠、科技 | 技术、金融、医疗 |
| 橙色 | 活力、热情、创新 | 销售、市场、创业 |
| 绿色 | 成长、稳健、可持续 | 运营、环保、教育 |
| 紫色 | 高贵、创意、前沿 | 品牌、设计、艺术 |
| 红色 | 紧迫、重要、警示 | 仅用于强调和警示 |

### HSB色彩空间
- **H（色相）**：0-360度，决定颜色种类
- **S（饱和度）**：0-100%，决定颜色鲜艳程度
- **B（明度）**：0-100%，决定颜色明暗

**高端PPT配色参数建议**：
- H：根据品牌/场景选择
- S：70-90%（鲜艳但不刺眼）
- B：80-95%（明亮但不失层次）
