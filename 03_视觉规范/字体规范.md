# 字体规范

> AI PPT导演系统标准字体使用指南
> 确保文字清晰、专业、现代

---

## 字体选择原则

### 1. 核心要求
- **清晰易读**：最小字号≥20pt，确保5米外能看清
- **现代专业**：使用无衬线字体（Sans-serif）
- **系统兼容**：优先使用系统自带字体，确保跨平台一致性
- **层次分明**：通过字号、字重、颜色区分层级

### 2. 禁用字体
❌ **避免使用**：
- 宋体、仿宋等衬线字体（不适合投影展示）
- 华文行楷、华文彩云等书法字体（不够专业）
- 过于花哨的装饰字体
- 字重过细的字体（＜Regular）

## 字体使用规范

### 1. 字号体系

| 元素 | 字号（pt） | 字重 | 颜色 | 使用场景 |
|------|-----------|------|------|---------|
| **封面主标题** | 72-96 | Bold | 渐变主色 | 封面页 |
| **封面副标题** | 36-48 | Medium | 深灰 | 封面页 |
| **章节标题** | 60-72 | Bold | 渐变主色 | 章节封面 |
| **页面标题** | 48-54 | Semibold/Bold | 深灰或主色 | 每页顶部 |
| **核心数据** | 60-96 | Bold | 主色 | 数据展示 |
| **小标题** | 32-36 | Semibold | 深灰 | 二级标题 |
| **正文** | 24-28 | Regular | 深灰 | 文字说明 |
| **注释/来源** | 20-22 | Regular | 中灰 | 图表注释 |

**注意**：
- 投影展示时，最小字号不得低于20pt
- 打印版本可适当缩小10-20%

---

### 2. 字重（Font Weight）使用

```
Bold（粗体）：大标题、核心数据
  - 字重值：700
  - 使用频率：10-15%的文字

Semibold（半粗）：页面标题、小标题
  - 字重值：600
  - 使用频率：15-20%的文字

Medium（中等）：重点正文、封面副标题
  - 字重值：500
  - 使用频率：20-30%的文字

Regular（常规）：正文、说明
  - 字重值：400
  - 使用频率：40-50%的文字
```

**原则**：同一页面不超过3种字重

---

### 3. 行距与段距

#### 标题
```
行距：1.2-1.3倍字号
段前距：0.5-1倍字号
段后距：0.5-1倍字号
```

#### 正文
```
行距：1.5-1.8倍字号
段前距：0.5倍字号
段后距：1倍字号
```

#### 数据
```
独立显示，周围留白≥2倍字号
```

---

### 4. 对齐方式

| 元素 | 对齐方式 | 说明 |
|------|---------|------|
| 封面标题 | 居中 | 营造仪式感 |
| 章节标题 | 居中或居左 | 根据版式选择 |
| 页面标题 | 居左 | 符合阅读习惯 |
| 正文 | 居左 | 提高可读性 |
| 核心数据 | 居中 | 突出重点 |
| 图表标题 | 居左或居中 | 与图表对齐 |

**原则**：避免右对齐和两端对齐（中文不适用）

---

### 5. 颜色使用

#### 标题颜色
```
方式1：纯色
- 使用主色（如#4A90E2）
- 适合小标题

方式2：渐变（推荐）
- 使用主色渐变（如#4A90E2 → #5C6BC0）
- 适合大标题、章节标题
- 渐变角度：0-180度

方式3：深灰
- 使用中性深灰（如#2C3E50）
- 适合页面标题（当页面已有彩色元素时）
```

#### 正文颜色
```
标准：深灰色
- 方案A：#2C3E50
- 方案B：#34495E
- 方案C：#2D3436

避免：纯黑（#000000）- 过于生硬
避免：浅灰（<#666666）- 对比度不足
```

#### 注释颜色
```
中灰色：#7F8C8D 或 #95A5A6
透明度：80-90%
```

## 特殊场景字体使用

### 1. 数据展示

#### 纯数字
```
字体：SF Pro Display / DIN / Roboto
原因：等宽设计，数字更整齐美观
字号：60-96pt
字重：Bold
颜色：主色或渐变
```

#### 数字+单位
```
数字：60-96pt，Bold
单位：40-50%数字字号，Regular
示例：
  5000 万元
  ^^^^  ^^
  60pt  30pt
```

#### 百分比/增长率
```
数字+符号：主色，Bold
示例：+150%、↑25%
```

---

### 2. 图表文字

#### 坐标轴标签
```
字号：20-24pt
字重：Regular
颜色：中灰（#7F8C8D）
```

#### 数据标签
```
字号：24-28pt
字重：Semibold
颜色：对应系列颜色或深灰
```

#### 图例
```
字号：22-24pt
字重：Regular
颜色：深灰
```

---

### 3. 表格文字

#### 表头
```
字号：26-28pt
字重：Semibold或Bold
颜色：白色（深色背景）或深灰（浅色背景）
背景：主色或深灰
```

#### 单元格内容
```
字号：24-26pt
字重：Regular
颜色：深灰
对齐：数字右对齐，文字左对齐
```

## 字体安装指南

### macOS安装
```bash
1. 下载字体文件（.ttf 或 .otf）
2. 双击字体文件
3. 点击"安装字体"
4. 重启PowerPoint/Keynote
```

### Windows安装
```bash
1. 下载字体文件
2. 右键字体文件 → "安装"
3. 或复制到 C:\Windows\Fonts\
4. 重启PowerPoint
```

## 常见问题

### Q1：为什么不用宋体？
**A**：宋体是衬线字体，适合印刷阅读，但在投影时笔画容易模糊。无衬线字体（如黑体）在大屏幕上更清晰。

### Q2：数字用什么字体最好看？
**A**：推荐使用等宽字体（如DIN、SF Pro、Roboto），数字对齐更整齐，视觉更专业。

### Q3：客户电脑没有我用的字体怎么办？
**A**：三种方案：
1. 导出为PDF（字体嵌入）
2. 将文字转为图片
3. 使用通用字体（微软雅黑/Arial）

### Q4：可以用艺术字吗？
**A**：不建议。高端PPT强调专业性，艺术字容易显得"廉价"。如需装饰，用图形元素而非特殊字体。

### Q5：英文全大写可以吗？
**A**：小标题可以用全大写增强设计感，但正文避免使用（降低可读性）。

## 检查清单

制作PPT时，请确认：
- [ ] 最小字号≥20pt
- [ ] 同一页面字体不超过2种（中英分开算）
- [ ] 字重不超过3种
- [ ] 标题与正文对比度明显（字号相差≥1.5倍）
- [ ] 颜色对比度符合可读性标准（≥4.5:1）
- [ ] 核心数据字号最大（60pt+）
- [ ] 行距合理（标题1.2-1.3，正文1.5-1.8）
