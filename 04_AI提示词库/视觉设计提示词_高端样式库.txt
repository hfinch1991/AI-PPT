提示词1:波浪线连接的卡片流程图
===========================================

Create a horizontal card flow diagram with wave connectors for a business presentation.

Design specifications:
- Layout: 4 cards arranged horizontally
- Connection: Smooth wave lines (~) connecting each card
- Cards alternating colors:
  Card 1: Deep blue #2C5282
  Card 2: Gold #D4AF37
  Card 3: Deep blue #2C5282
  Card 4: Gold #D4AF37

Card content structure:
- Top: Bold title (12-14pt)
- Middle: 3 bullet points
- Bottom: Key metric or data

Wave connector design:
- Smooth S-curve between cards
- Line width: 2-3pt
- Color: Gradient from previous card to next card
- Style: Flowing, professional, not cartoonish

Visual style:
- Modern, clean, corporate
- Minimal shadows (subtle drop shadow only)
- Rounded corners on cards (8-12px radius)
- Ample white space between elements
- 16:9 aspect ratio

Technical specs:
- PNG with transparent background
- Resolution: 1920x1080px
- Cards occupy center 70% of canvas
- 30% white space for PPT text overlay

提示词2:横向时间轴(彩色时间段标注)
===========================================

Design a horizontal timeline with colored time period blocks for project milestones.

Timeline structure:
- Horizontal line spanning 80% of width
- 4 time periods with different colored blocks above the line
- Each period marked with date range and key activities

Color coding for periods:
Period 1 (Jul): Deep blue #2C5282
Period 2 (1-3月): Gold #D4AF37
Period 3 (5月): Deep blue #2C5282
Period 4 (6月): Deep red-brown #8B4513

Each time block contains:
- Time label (top): "7月1日-12月"
- Key activities: 2-3 bullet points
- Achievement metric: Large number or percentage

Bottom data cards:
- 4 summary metric cards below timeline
- Format: "4个" "25%" "32%" "100%"
- Arranged horizontally with labels
- Minimal card design, icon optional

Visual requirements:
- Clean, minimalist design
- No decorative elements
- Professional corporate aesthetic
- Clear hierarchy: time > activities > metrics
- 16:9 aspect ratio, PNG format

提示词4:菱形对比图(问题vs对策)
===========================================

Design a diamond comparison layout showing problems vs solutions.

Layout structure:
- Center: Diamond shape with "存在问题 ⟷ 解决方案"
- Left side: 3 problem cards (deep blue #2C5282)
- Right side: 3 solution cards (gold #D4AF37)
- Arrows pointing from center to cards

Problem cards (left):
Card format:
┌─────────────────┐
│1. 跨部门协作效率不足│
│ 问题描述:XXX      │
└─────────────────┘

Solution cards (right):
Card format:
┌─────────────────┐
│1. 提高跨部门响应度│
│ 措施:建立XX机制   │
└─────────────────┘

Visual requirements:
- Cards aligned vertically, evenly spaced
- Problems and solutions on same horizontal level
- Diamond in center with gradient fill
- Subtle connecting lines/arrows
- Clean, professional, symmetrical
- No excessive decoration

Color scheme:
- Left cards: Deep blue background, white text
- Right cards: Gold background, dark text
- Center diamond: Gradient blue to gold
- Arrows: Neutral gray

Technical specs:
- 16:9 aspect ratio
- Center diamond 20% width
- Left cards 35%, right cards 35%, margins 10%
- PNG format, high resolution

提示词6:大数字展示页(成果卡片)
===========================================

Design a key metrics showcase page with large numbers and cards.

Page layout:
- Title at top: "2025年核心成果一览"
- 3-5 metric cards in grid layout
- Each card contains: Big number + growth rate + description

Card structure:
┌────────────────────┐
│    3000万          │  ← 48-60pt, bold, gradient color
│    ↑150%           │  ← 24pt, with up arrow
│  创历史新高         │  ← 18pt, description
└────────────────────┘

Color application:
- Odd cards: Deep blue gradient #1E3A5F → #2C5282
- Even cards: Gold gradient #D4AF37 → #E5BF4F
- Growth indicators: Green #00D084 (positive), Red #FF6B6B (negative)

Visual effects:
- Subtle card shadows
- Gradient backgrounds for numbers
- Minimalist borders
- Ample padding inside cards

Layout options:
Option 1: 2x2 grid (4 metrics)
Option 2: 3x1 top row + 2x1 bottom row (5 metrics)
Option 3: Asymmetric layout with one hero card

Background:
- White base
- Optional: 20-30% opacity gradient decoration in corners
- No distracting patterns

提示词8:数据卡片组(底部汇总指标)
===========================================

Design a set of summary data cards for bottom of timeline or overview page.

Card set structure:
- 4 cards arranged horizontally
- Equal width, minimal gaps
- Compact, information-dense

Individual card format:
┌─────────┐
│  [图标]  │
│   14     │  ← Large number
│  项目数   │  ← Label
└─────────┘

Data examples:
Card 1: "4个" - 某项指标
Card 2: "25%" - 增长率
Card 3: "32%" - 完成率
Card 4: "100%" - 达成率

Visual design:
- Minimal borders or no borders
- Icon at top (optional, simple line icon)
- Number: 36-48pt, bold
- Label: 14-16pt, regular
- Background: White or very light gray
- Accent: Thin colored top border (matches theme color)

Color coding:
- Can use single theme color for all
- Or: Different pastel shades per card
- Keep it subtle, not overwhelming

Use cases:
- Bottom summary for timeline pages
- Footer metrics for overview pages
- Quick stats snapshot

提示词9:渐变背景装饰元素
===========================================

Create subtle gradient background decoration elements for slides.

Element types:

Type 1: Corner gradient blob
- Placement: Top-right or bottom-left corner
- Shape: Organic, flowing blob shape
- Colors:
  方案D: #1E3A5F to #2C5282, 20% opacity
  方案E: #2C5282 to #3A6FA5, 25% opacity
- Coverage: 15-20% of slide area
- Edge: Soft blur, no hard edges

Type 2: Diagonal gradient band
- Placement: From corner to opposite side
- Width: 30-40% of slide width
- Colors: Two-color gradient from scheme
- Opacity: 15-20%
- Angle: 45° or 135°

Type 3: Abstract geometric pattern
- Elements: Circles, lines, or hexagons
- Arrangement: Scattered or grid-based
- Colors: Single color from scheme
- Opacity: 10-15%
- Coverage: Background layer, very subtle

Requirements for all:
- Must not interfere with text readability
- Low opacity (10-30% max)
- Complement, not compete with content
- Professional, not playful
- PNG with transparency

提示词10:高端封面背景
===========================================

Design a premium cover page background for annual report presentation.

Visual concept:
- Diagonal or radial gradient as base
- Abstract geometric elements overlaid
- Space for centered title (50% vertical space)
- Professional lighting effect

Gradient options:

方案D (高端金蓝):
- Base gradient: #1E3A5F (bottom-left) to #2C5282 (top-right)
- Accent elements: Gold #D4AF37 geometric shapes (10% opacity)
- Mood: Sophisticated, executive-level

方案E (多层次专业):
- Base gradient: #2C5282 (bottom) to #3A6FA5 (top)
- Accent layer 1: Orange #FF9500 (5% opacity, corner)
- Accent layer 2: Gold #D4AF37 thin lines or circuits
- Mood: Comprehensive, strategic

Geometric elements:
- Hexagons, circles, or abstract lines
- Semi-transparent (10-20% opacity)
- Subtle 3D depth (optional slight shadow)
- Modern tech aesthetic

Lighting:
- Top-right light source simulation
- Subtle gradient highlights
- Professional studio lighting feel

Text space:
- Center 50% kept relatively clear
- Enough contrast for white or gold text
- No busy patterns in text area

Technical:
- 16:9 aspect ratio
- High resolution (1920x1080 minimum)
- PNG format
- Optimized for projection display

===========================================
质量检查清单
===========================================

生成的视觉元素应符合:
□ 背景透明或纯白(便于叠加到PPT)
□ 不包含任何实际文字(仅占位框)
□ 配色符合预设方案(A/B/C/D/E之一)
□ 风格一致(同一PPT的所有元素)
□ 简约大于复杂(高端=简洁+质感)
□ 高分辨率(1920x1080或更高)
□ 适合投影展示(对比度合理,不过暗或过亮)
□ 具体样式明确(波浪线/时间轴/半圆/菱形/环形等)

===========================================
常见问题及解决方案
===========================================

Q:如何确保波浪线连接自然流畅?
A:在提示词中强调"smooth S-curve""flowing wave""natural connector",避免"zigzag"或"angular"

Q:时间轴的颜色块如何区分但不冲突?
A:使用同一色系的不同明度,或使用配色方案E的多颜色(蓝/橙/金/褐),确保都是专业色

Q:菱形对比图如何保持对称美?
A:明确要求"symmetrical layout""evenly spaced""aligned horizontally"

Q:如何让环形展示不显得凌乱?
A:强调"organized""orbital layout""clear spacing""ample white space in center"
