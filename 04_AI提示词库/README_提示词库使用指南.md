## 🎯 使用建议

**推荐路径**:基础版 → 增强版

1. 先阅读基础版,理解通用概念
2. 参考增强版的具体案例,理解"好的提示词"长什么样
3. 根据自己的场景,复制修改增强版提示词

1. 根据PPT场景,选择对应的快速生成模板
2. 填入自己的数据,直接运行
3. 根据输出调整细节

### 1. 结构化生成提示词_6章节模板.txt

**核心价值**:提供经过验证的6章节黄金结构(15%-30%-20%-15%-10%-10%)

**包含内容**:
- ✅ 完整6章节PPT大纲生成提示词
- ✅ 单页面详细内容生成提示词
- ✅ 5种经过验证的可视化样式(波浪线/时间轴/菱形图/半圆图/环形图)

**适用场景**:

- 15-20页的中长篇汇报

**示例提示词**:
```

提示词7:快速生成 - 工作成果展示型(14页)
提示词8:快速生成 - 年度个人述职型(16页)

**关键特点**:
- 明确指定章节权重(第2章必须占30%)
- 提供具体可视化建议(不只说"图表",而是"波浪线连接的卡片流程")
- 包含配色建议(深蓝#2C5282/金色#D4AF37等)

**核心价值**:提供专门针对PPT展示的数据分析方法

**包含内容**:
- ✅ PPT级别的关键洞察提炼
- ✅ 时间轴数据叙事生成
- ✅ 对比分析数据生成(实际vs目标)
- ✅ 结构占比数据分析
- ✅ 增长归因分析
- ✅ 核心成果数据包生成
- ✅ 问题识别与量化
- ✅ 目标设定数据生成
- ✅ 数据故事完整叙事
- ✅ 快速数据整理

**适用场景**:
- 将原始数据转化为PPT展示内容
- 需要量化表达、视觉友好的数据呈现
- 数据密集型PPT

**示例提示词**:
```
提示词1:提炼PPT级别的关键洞察
输出格式:

关键数据:Q4 GMV 3000万
增长率:同比增长150%,环比增长35%
业务意义:突破全年目标,占全年总营收42%
推荐可视化:柱状图(Q1-Q4对比)+大数字展示
PPT标题建议:"Q4业绩3000万,创历史新高"
```

**关键特点**:
- 所有数据必须量化(不说"显著增长",要说"增长150%")
- 每个洞察都有推荐的可视化方式
- 提供直接可用的PPT页面标题

## 💡 使用技巧

### 技巧1:提示词的迭代优化

**第一轮**:使用基础提示词生成初稿
```
使用提示词1生成大纲
```

**第二轮**:追加具体要求
```
"请优化第3页,增加量化数据"
"请为第5页补充具体案例"
```

**第三轮**:根据反馈调整
```
"请将第2章的页数从3页增加到5页"
"请将配色从蓝色改为金蓝配色"
```

---

### 技巧2:组合使用多个提示词

**示例:生成一页完整的"核心成果展示"页**

Step 1: 使用 `数据分析提示词_PPT导向.txt` → 提示词1
```
提炼3个核心成果:
1. GMV 3000万,↑150%
2. 新客户500家,↑80%
3. 满意度92%,行业第一
```

Step 2: 使用 `图表生成提示词_高端图表库.txt` → 提示词1
```
生成大数字展示卡片图(3个指标)
```

Step 3: 使用 `视觉设计提示词_高端样式库.txt` → 提示词9
```
生成渐变背景装饰元素
```

Step 4: 组装到PPT
- 插入生成的图表为背景
- 添加标题:"2025年核心成果:全面超额完成"
- 调整配色和字体

---

### 技巧3:自定义提示词模板

**基于现有提示词创建自己的模板**:

1. 选择一个接近的提示词(如提示词7)
2. 复制到新文件
3. 修改数据部分,保留结构部分
4. 保存为"我的_XXX提示词.txt"
5. 下次直接使用

**示例**:
```
原提示词:提示词7 - 快速生成工作成果展示型(14页)
↓
我的提示词:快速生成销售部季度汇报(12页)
- 修改:章节权重调整(强调业绩数据)

## ❓ 常见问题

### Q1:基础版和增强版有什么区别?
**A**:
- 基础版:通用指导,适合理解概念
- 增强版:具体案例,包含可运行代码和详细设计样式

### Q2:我是新手,应该从哪个开始?
**A**:
- 建议路径:先看基础版理解概念 → 再用增强版实践

### Q3:提示词太长,AI理解不了怎么办?
**A**:
- 分步提问:将长提示词拆解为多个小步骤
- 提供示例:给AI一个"好的案例"作为参考

### Q4:生成的内容不符合预期怎么办?
**A**:
- 追加具体要求:"请优化第3条""换个角度"
- 提供更多背景信息
- 使用"一句话总结"要求AI精简

### Q5:如何确保多个AI生成的内容风格一致?
**A**:
- 在所有提示词中使用相同的配色方案
- 在所有提示词中强调"高端商务风格""简约专业"
- 使用相同的视觉元素参考(如"波浪线连接""时间轴")

### Q6:Python代码运行报错怎么办?
**A**:
- 检查是否安装了matplotlib/numpy等依赖
- 检查中文字体是否安装
- 查看错误信息,通常是字体或数据格式问题

### Q7:生成的图表能直接用于印刷吗?
**A**:
- 能。所有图表默认DPI=300,符合印刷标准
- 如需更高清,可以调整figsize和dpi参数
